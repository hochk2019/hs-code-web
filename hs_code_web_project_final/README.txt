
# HS Code Web Project (Final)

## Mô tả
Web hỗ trợ tra cứu mã HS code và cập nhật chính sách nhập khẩu tại Việt Nam. Có phân quyền đăng nhập và tích hợp AI hỗ trợ tìm kiếm thông minh.

## Chức năng nổi bật
- Tra cứu HS code
- Gợi ý mã HS bằng AI
- Quản lý bài viết chính sách, bình luận
- Nạp dữ liệu Excel, tải thông tư/nghị định từ file hoặc link
- Phân tích mô tả HS để tự sinh từ khóa

## Cách chạy
1. Cài thư viện:
    pip install -r requirements.txt
2. Chạy:
    python main.py
3. Truy cập:
    http://127.0.0.1:5000
